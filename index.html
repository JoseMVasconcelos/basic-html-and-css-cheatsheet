Basic HTML Tags

Tag Tag Name Description
<html> HTML Tag Defines an HTML document.
<head> Head Contains the metadata for the document.
<title> Title Defines a title for the document.
<body> Body Defines the document`s body.
<h1...h6> Header 1, 2... 6 Defines HTML Headers.
<p> Paragraph Defines a simple paragraph in an HTML page.
<br> Break Line Inserts a single line break in an HTML page.
<hr> Horizontal Rule Defines a thematic break in an HTML page.
<!--...--> Comment Inserts comments in the source code.

Links/Content Tags

Tag Tag Name Description
<a> Anchor Defines an Hyperlink in an HTML page.
<link> Link Defines the relationship between a document and an external resource.
<img> Image Defines an image in an HTML page.
<figcaption> Figure Caption Defines a caption for a <figure> element.
<figure> Figure Specifies self-contained content
<audio> Audio Defines sound content
<video> Video Defines a video or movie
<source> Source Defines multiple media resources for media elements

List Tags

Tag Tag Name Description
<ul> Unordered List Defines an unordered list in an HTML page.
<ol> Ordered List Defines an ordered list in an HTML page.
<li> List Item Defines a list item inside a un/ordered list.

Table Tags

Tag Tag Name Description
<table> Table Defines a table in an HTML page.
<thead> Table Head Groups the header content in a table.
<tbody> Table Body Groups the body content in a table.
<tfoot> Table Foot Groups the footer content in a table.
<th> Table Header Defines a header cell in a table.
<tr> Table Row Defines a row in a table.
<td> Table Data Defines a cell in a table.

Basic HTML Attributes

Attribute Tag Name Description
href Hypertext Reference <a> attribute, specifies the URL of the page the link goes to.
src Source Specifies the path to the image/video to be displayed.
width Width <image> attributes, provides size information for images.
height Height
alt Alternative <image> attribute, provides an alternate text for an image.
lang Language <html> attribute, declares the language of the Web page.
colspan Column Span Defines how many columns a <td> shoud span.
rowspan Row Span Defines how many rows a <td> shoud span.

CSS Color Properties

Property Attribute Description
color Name or code of the desired color Defines a color for the object.
background-color Name or code of the desired color Defines a color for the backgroud of the object.

CSS Font Properties

Property Attribute Description
font-family Name of the desired font Changes the font of the desired text.
font-style normal, italic, oblique Mostly used to specify italic text.
font-size 1...1000px Changes the size of the font.
